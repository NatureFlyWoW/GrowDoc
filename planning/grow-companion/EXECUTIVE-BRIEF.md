# GrowDoc: Grow Companion — Executive Brief

**Prepared for:** Product Leadership  
**Date:** 2026-04-08  
**Duration:** 3-minute read

---

## THE OPPORTUNITY

GrowDoc has a solid diagnostic tool (Plant Doctor v3). Now we expand into a **daily companion** that growers use throughout their entire grow—not just when problems arise.

**Market size:** ~1M indoor cannabis growers in North America  
**Addressable market:** 5–10K users in Year 1 (0.5–1% penetration)  
**Revenue potential (Year 2+):** $20–50K/month at 5% freemium conversion

---

## THE INSIGHT

**Current state:** Users visit Plant Doctor once when their plant is sick.  
**Desired state:** Users visit daily to check tasks, monitor progress, log outcomes.

**Why this matters:** Daily habit tools have 3–5x better retention than one-shot tools. They also build defensible network effects (community intelligence) that competitors can't replicate.

**The moat:** Our outcome data becomes the moat. When 1,000+ growers verify "Soil + LED users with CalMag deficiency have 92% success rate," that's competitive advantage money can't buy.

---

## THE STRATEGY

### Phase 1: Foundation (4 weeks) — "Grow Intelligence"
**Goal:** Transform Plant Doctor into a proactive daily companion.

**Core features:**
1. **Enhanced Setup Profile** — Space, nutrients, climate control (enables all personalization)
2. **Growth Timeline** — Visual progression with stage milestones (proactive guidance)
3. **Daily Task List** — Smart actions based on stage + setup (habit formation)
4. **Environment Calculator** — VPD/DLI/EC targets (removes guesswork)
5. **Outcome Verification** — "Did this work?" auto-prompts (closes feedback loop)

**Effort:** ~25 dev days | **Deployment:** Vercel (Week 4)

**Success metrics (Month 1 post-launch):**
- >70% 7-day retention (vs ~20% current)
- >40% daily task completion rate
- >50% setup profile adoption

### Phase 2: Moat (8 weeks) — "Outcome Loops"
**Goal:** Close feedback loops so data improves recommendations.

**Features:** Enhanced journal, harvest window advisor, outcome aggregation

**Expected impact:** 30-day retention +15%, success rate badges in diagnoses

### Phase 3: Premium (8 weeks) — "Goal-Driven Personalization"
**Goal:** Personalize by grower's objectives (yield vs quality vs terpenes).

**Features:** Priority sliders, strain database, training schedules, advanced analytics

**Expected impact:** Premium tier adoption 5%+ = $20K+/month revenue

## RISKS & MITIGATIONS

| Risk | Impact | Mitigation |
|------|--------|-----------|
| **Low retention (<30% D7)** | Unit economics fail | A/B test task language; make completion frictionless; monthly user interviews |
| **Grow with Jane adds diagnosis** | Feature parity threat | Ship Phase 1 fast (4 weeks); establish moat with outcome data |
| **Task completion <30%** | Habit loop doesn't form | Redesign task language; test optimal daily task count; track completion metrics |
| **Outcome verification <50%** | Feedback loop broken | Use smart timing; make check-in non-intrusive; show value ("Success rate badge") |
| **Privacy concerns** | User hesitation on data sharing | Transparent opt-in; anonymized aggregation; local-first by default |
| **Franco availability** | Key person risk | Document all protocols; build Franco's IP library now; develop secondary expert (future) |
