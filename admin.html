Sign in with GitHub

You need a GitHub Personal Access Token with public_repo scope to edit documents. Your token is stored only in your browser.

  1. Go to github.com/settings/tokens/new
  2. Scope public_repo is pre-selected. Click "Generate token"
  3. Copy the token (starts with ghp_) and paste it below
