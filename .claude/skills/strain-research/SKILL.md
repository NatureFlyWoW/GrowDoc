# Strain Research Expertise

## Core Principle

Strain selection is the single highest-leverage decision in a grow. Everything downstream is constrained by the genetics you start with. Research before purchasing saves months of growing the wrong plant.

---

## Research Sources (reliability-ranked)

1. **Seedfinder.eu** — Most comprehensive lineage database. Cross-reference ALL breeder claims.
2. **Phylos Galaxy / Kannapedia** — Genetic testing data when available. Objective genotype.
3. **ICMag grow journals** — Multi-harvest reports from experienced growers. Filter for indoor, similar wattage.
4. **Grower.ch** (German) — European-focused, Austrian/German/Swiss conditions.
5. **Rollitup strain reviews** — Large sample size, variable quality. Look for photos + detail.
6. **Overgrow** — Strong on landrace and heritage genetics.
7. **Breeder's direct info** — Treat as marketing. Never trust flowering times or THC claims.

### Red Flags

- Flowering time claims: add 1–2 weeks to breeder-stated time. Under moderate LED PPFD (400–600 µmol/m²/s), add another 3–7 days beyond HPS-era forum reports.
- THC > 30%: extremely rare outside lab-optimized conditions
- New breeder with no grow reports: unknown stability, possible hermie tendency
- Repackaged genetics: check lineage on Seedfinder

## Lineage Analysis Framework

Map family tree 2+ generations deep. Extract:

| Question | Source |
|---|---|
| Terpene dominance | Both parents' known profiles |
| Stretch ratio | Indica/sativa ratio + parent grow reports |
| Realistic flowering time | Longer parent + 5–10 day buffer |
| Mold resistance | Landrace heritage (tropical = prone, Afghan = resistant) |
| Hermie tendency | Breeder rep + parent history (Cookies lineage carries risk) |
| Effect profile | Both parents' effects, weighted by dominance |

### Common Genetic Building Blocks

| Base | Brings | Watch For |
|---|---|---|
| **OG Kush** | Gas terps, potency, caryophyllene | Hungry, stretch, PM |
| **GSC** | Complex terps, bag appeal | Slow start, hermie risk |
| **Skunk #1** | Vigor, yield, reliability | Can lack complexity, loud smell |
| **Haze** | Cerebral, creative, long-lasting | 12–16wk flower, extreme stretch |
| **Afghan/Kush** | Short, dense, fast, resinous | One-dimensional, couch-lock |
| **Chem/Diesel** | Fuel terps, potency, unique | Slow veg, stinky, hermie under stress |
| **Cherry Pie** | Sweet/tart terps, structure, colors | Moderate potency, needs full cure |
| **Durban Poison** | Sativa energy, terpinolene | Extreme stretch, long flower |

## Strain Matching Protocol

### Step 1 — Target Profile
```
EFFECT:      [cerebral/balanced/body] [energizing/neutral/sedating]
TERPENES:    [target profile or aroma description]
GROW TRAITS: [max height / flower weeks / yield priority / training response]
ENVIRONMENT: [tent size / wattage / temp / humidity]
STEALTH:     [smell tolerance / speed priority]
```

### Step 2 — Hard Constraint Filter
Eliminate: height violations, flower time too long, smell too intense for stealth, feed sensitivity mismatch.

### Step 3 — Score candidates (1–10 weighted, same framework as pheno-hunting)

### Step 4 — Source Verification
- Lineage confirmed on Seedfinder.eu
- 3+ independent grow reports
- Breeder reputation check
- Seed type matches plan (fem/reg/auto)
- Local shop availability (OPSEC)

## Crossing Project Evaluation

| Factor | Good Sign | Bad Sign |
|---|---|---|
| Terpene complementarity | Shared + unique terps | Identical profiles |
| Structure balance | Tall + compact | Both extreme |
| Flowering time | Within 2 weeks | 8wk × 14wk |
| Potency | Both 18%+ | One is filler |
| Stability | Both IBL or stable F1 | Both unstable polyhybrids |

### Cross Types → Expected Outcomes

| Cross | F1 | F2 | Best For |
|---|---|---|---|
| IBL × IBL | Uniform, vigorous | Wide variation | Targeted trait combo |
| F1 × F1 | Moderate variation | Extreme variation | Max diversity |
| BX | Leans to recurrent parent | Tighter | Recovering traits |
| S1 | Narrows to dominant | Further narrowing | Preserving without clone |

**Autoflowers:** Not covered here — cannot be cloned, not suitable for pheno-hunting, fixed lifecycle limits optimization.

## Breeder Evaluation Checklist

Static tier lists age fast. Instead, evaluate any breeder against:

- **Years active:** 5+ years = established track record
- **Award history:** Cannabis cups, community recognition
- **Genetic testing published:** Phylos/Kannapedia verified = high confidence
- **Grow report volume:** 10+ independent journals on ICMag/Grower.ch = well-tested
- **Hermie complaint rate:** Search "[breeder] hermie" on forums. >5% complaint rate = red flag
- **Seed consistency:** Multiple growers reporting similar phenotype range = stable genetics

**Illustrative examples (as of 2025–2026):**
- High confidence: Bodhi Seeds, Archive, CSI Humboldt (decades active, genetic-tested, extensive grow reports)
- Solid middle: Sensi Seeds, Dutch Passion, T.H. Seeds (long history, occasional variation, well-documented)
- Approach with caution: Instagram-only breeders, white-label repackagers, unnamed "house genetics"

These are snapshots — always re-evaluate with the checklist above.

## Research Workflow

1. Define target profile
2. Generate 5–10 candidates from knowledge + web research
3. Deep-dive lineage on top 3
4. Cross-reference with grow reports (include German-language forums)
5. `/devils-advocate` before final recommendation
6. Verify sourcing (local availability, seed type, price)
